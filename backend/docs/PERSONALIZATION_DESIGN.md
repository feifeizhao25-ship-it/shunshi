# 顺时 · 个性化推荐系统设计文档

> 日期: 2026-03-19
> 目标: 所有UI围绕用户个性化展开，知识库用AI图片完善

## 一、系统架构

```
用户画像 (User Profile)
├── 体质类型: 气虚/阳虚/阴虚/痰湿/湿热/血瘀/气郁/特禀/平和
├── 健康状态: 睡眠/运动/情绪/饮食 记录
├── 当前节气: 惊蛰 (spring)
├── 季节: 春季
└── 年龄阶段: 青年/中年/老年

推荐引擎 (后端 API)
├── 综合因子: 体质 × 节气 × 健康状态 × 季节
├── 排序: 相关性评分 (0-100)
└── 输出: 每个维度 TOP-N 推荐内容
    ├── 今日推荐 (首页)
    ├── 节气养生 (节气详情)
    ├── 体质调理 (体质结果)
    ├── 食疗方案 (养生Tab)
    ├── 茶饮推荐 (养生Tab)
    ├── 运动建议 (养生Tab)
    ├── 穴位按摩 (养生Tab)
    └── 睡眠改善 (养生Tab)
```

## 二、个性化推荐算法

### 评分权重

| 因素 | 食疗 | 茶饮 | 运动 | 穴位 | 睡眠 | 情绪 |
|------|------|------|------|------|------|------|
| 体质匹配 | 40 | 35 | 30 | 25 | 20 | 15 |
| 节气匹配 | 30 | 30 | 25 | 10 | 15 | 10 |
| 季节匹配 | 15 | 15 | 25 | 10 | 15 | 10 |
| 健康状态 | 10 | 10 | 15 | 20 | 40 | 55 |
| 用户偏好 | 5 | 10 | 5 | 35 | 10 | 10 |

### 体质 × 内容 映射规则

```
气虚质 → 补气类(黄芪/人参/山药)、健脾食材、舒缓运动
阳虚质 → 温阳类(羊肉/生姜/桂圆)、艾灸、温补运动
阴虚质 → 滋阴类(银耳/百合/枸杞)、静心运动(瑜伽/太极)
痰湿质 → 祛湿类(薏米/赤豆/冬瓜)、有氧运动
湿热质 → 清热类(绿豆/苦瓜/冬瓜)、游泳
血瘀质 → 活血类(山楂/红花/玫瑰)、舒筋运动
气郁质 → 疏肝类(玫瑰花/佛手/柑橘)、户外运动
特禀质 → 固表类(黄芪/大枣/灵芝)、温和运动
平和质 → 四季时令食材、均衡运动
```

## 三、知识库完善 (image_url)

### 方案: AI生成 + 本地存储

1. 为每种内容类型设计图片生成 Prompt
2. 用 SiliconFlow/AI 生成图片
3. 存储到 `/static/images/contents/` 目录
4. 数据库新增 `image_url` 字段
5. API 返回图片 URL

| 内容类型 | 图片风格 | 数量 |
|---------|---------|------|
| 食疗 (110) | 中式美食摄影、俯拍、暖色调 | 110张 |
| 茶饮 (45) | 茶具特写、柔和光线 | 45张 |
| 运动 (50) | 运动场景、人物剪影 | 50张 |
| 穴位 (42) | 人体穴位图、简约风格 | 42张 |
| 睡眠 (30) | 安静卧室、暖光 | 30张 |
| 情绪 (28) | 自然风景、治愈系 | 28张 |

## 五、后端 API 设计

### 新增 API

```
POST /api/v1/recommend/personalized
Body: {
  constitution_type: "qixu",     // 体质
  health_status: { sleep: 72, exercise: 45, mood: 68 },
  season: "spring",               // 季节
  solar_term: "惊蛰",             // 节气
  categories: ["food_therapy", "tea", "exercise"],
  limit: 5
}
Response: {
  recommendations: [
    {
      id: "...",
      title: "菠菜养肝汤",
      type: "food_therapy",
      image_url: "/static/images/contents/food_001.jpg",
      match_score: 92,
      match_reasons: ["适合气虚质", "春季养肝"],
      description: "...",
      difficulty: "简单",
      duration: "20分钟",
      ingredients: [...],
      steps: [...]
    }
  ]
}

GET /api/v1/contents/{id}
→ 完整内容详情 (含图片、步骤、功效)

POST /api/v1/contents/{id}/like
POST /api/v1/contents/{id}/view
```
